# 火柴人冒险 - 横版游戏

这是一个使用Vue 3和Tailwind CSS构建的简单火柴人横版冒险游戏。

## 功能特性

- SVG绘制的火柴人角色，有跑步、跳跃和站立动画
- 物理引擎包括重力和碰撞检测
- 平台跳跃
- 敌人与生命系统
- 得分系统

## 安装与运行

### 安装依赖
```bash
npm install
```

### 开发环境运行
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 游戏控制

- 左右箭头键或A/D: 移动
- 上箭头键、W或空格: 跳跃

## 技术栈

- Vue 3 + TypeScript
- Tailwind CSS
- Vite
